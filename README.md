# Playdio

## An aesthetic web radio player powered by the Radio Browser API.

Playdio is a minimalist internet radio player focused on creating a calm listening experience. Instead of copying the look of modern streaming services, it embraces a warm, vinyl-inspired interface with dynamic theming that adapts to the currently playing station.

---

## ✨ Features

- 📻 Browse thousands of internet radio stations
- 🔍 Instant station search
- 💿 Vinyl-inspired player interface
- 🎨 Dynamic color themes generated from station artwork
- 🔊 Built-in volume controls
- 📡 Powered by the Radio Browser API
- ⚡ Lightweight and responsive
- 🌙 Minimal, distraction-free design
